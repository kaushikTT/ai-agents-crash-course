# Codebase Archaeology Outputs

This folder is managed by the **Codebase Archaeology Agent** (Kiro global agent).

| File | Description |
|---|---|
| `*-archaeology-report.md` | Structured codebase intelligence report |
| `*-archaeology-run.log`   | Agent execution log for the corresponding run |

Do not manually edit files in this folder. Re-run the agent to refresh the analysis.
